#  Bootstrap前端框架简介

bootstrap中文网站：https://code.z01.com/bootstrap/docs/


新增模板后的目录结构如下：

```
├── app
│   ├── __init__.py
│   ├── models.py
│   ├── routes.py
│   └── templates
│       ├── about.html
├── manage.py
├── migrations
└── venv
```
