Python搭建开发环境

本节我们来在mac系统和windows系统安装Python开发环境和安装Pycharm开发工具。

Win10系统下载安装Python3.9

本节我们将向大家介绍如何在Win10系统安装 Python3.9 开发环境。Python 3.9使用了一个新的解析器，我们将在Python 3.10及更高版本中开始使用这种灵活性。

下载安装Python3.9环境

打开Python 官网：https://www.python.org/ 首先把鼠标悬浮在Downloads按钮上，其次点击Windows按钮如下

进入到下载页面，我们找到Python 64位安装文件，点击下载，如下图所示

双击我们下载好的Python64位文件，在点击运行开始安装Python

在弹出的界面窗口，勾选Add Python to Path，勾选Add Python to Path，勾选Add Python to Path重要的事情说3遍

勾选完成以后，单击用户自定义安装

在弹出的界面中，我们直接单击next按钮，如下图所示

在弹出的界面中，我们单击Browse按钮，改变我们的安装路径

笔者这里选择F盘，小伙伴可以随意选择，接下来单击左下角的新建文件夹，命名为Python3.9，最后一步单击“确定按钮”。

在弹出的窗口，单击intall按钮

安装中，直到弹出下面的窗口，说明小伙伴们的Python3.9已经安装成功了

-Win10检查Python3.9是否安装成功

首先单击屏幕左下角的放大镜图标，在弹出的搜索栏中输入cmd，单击命令行提示符，如下图所示

在弹出的窗口中，输入python，敲击回车键，如果小伙伴们出现的问题和下图一样，那么恭喜你，你的Python安装已经成功了！

macOS系统下载安装Python3.9

本节我们将向大家介绍如何在macOS系统安装 Python3.9 开发环境。在macOS系统安装Python非常简单，

-下载安装Python3.9环境

打开Python 官网：https://www.python.org/ 首先把鼠标悬浮在Downloads按钮上，其次点击Python3.9.2按钮，下载Python开发环境如下图：

找到下载好的Python3.9.2，双击打卡它

接下来进入傻瓜式操作，在弹出的窗口中单击继续按钮

在弹出的新的窗口中单击继续按钮

在新的窗口中单击继续按钮

在弹出的新的窗口中单击同意按钮

在弹出的新的窗口中单击安装按钮

在弹出的新的窗口中先输入我们mac系统中设定的密码，然后单击安装软件

等待一小会，会弹出安装成功的界面，如下图所示

-mac检查Python3.9是否安装成功

我们按住command+空格键，在搜索栏中输入terminal，进入终端

在弹出的界面输入python3.9，敲击回车键，如果安装成功，会弹出下面的提示文字。

Win10下载安装Pycharm

本节我们将向大家介绍如何在Win10系统下载安装 Pycharm。PyCharm是一种Python 编辑器，它是众多Python编辑器中比较好的一个。而且可以跨平台，在macos和Windows系统中都可以用。

-下载安装Pycharm

Window版Pycharm 下载链接：https://www.jetbrains.com/pycharm/download/#section=windows 首先鼠标单击上面的链接，跳转页面如下

选择windows选项卡，选择Pycharm社区版，单击下载按钮

找到我们下载到本地磁盘的Pycharm，双击它，在弹出的窗口单击运行按钮

在弹出的窗口单击Next按钮 在弹出的窗口单击Browse按钮，选择自己喜欢的安装路径，然后单击Next按钮 在弹出的窗口把能勾选的位置全部勾选上，然后单击Next按钮 在弹出的窗口单击Install按钮 在弹出的窗口选择第二个：稍后重启，单击Finish按钮 回到我们的桌面，找到Pycharm生成的快捷方式，双击它 在弹出的窗口，勾选同意协议，单击Continue按钮 在弹出的窗口，选择不发送数据，单击Don't send

Pycharm创建项目及文件​

在弹出的窗口中，选择创建新的项目 在弹出的窗口中，更改创建项目的位置，然后单击Create按钮来创建我们的第一个项目 在弹出的窗口中，鼠标悬浮到studyPython上，然后右键选择new，选择Directory来创建一个文件夹 在弹出的窗口中输入文件夹的名称，单击OK按钮 把鼠标移动到左侧树状目录的chapter1上，然后右键选择new，选择Python file来创建一个Python文件 在弹出的窗口中输入Python文件的名称，单击OK按钮 下面展示Pycharm编辑的主要的两个窗口

运行Python文件​

在编辑代码空白区单击右键，在弹出的窗口中选择run‘hello’.py来运行Python文件 运行结果如下图

设置Pycharm字体​

第一次打开Pycharm，代码默认字体会很小，为了保护我们的视力，我们先来调整一下代码字体的大小，首先我单击菜单栏的file按钮，找到设置settings，单击settings， 在弹出的窗口中展开Editor，单击下面的Font按钮，选择自己喜欢的字体和字体的大小，然后在单击Apply，最后在单击OK，这样我们的字体就设置成功了

设置Pycharm主题​

首先我单击菜单栏的file按钮，找到设置settings，单击settings

在弹出的窗口中展开Editor，展开Color Scheme，单击下面的genral按钮，在右侧选择自己喜欢的主题，然后在单击Apply，最后在单击OK，这样我们的Pycharm主题就设置成功了

更改Python解释器​

本节我们来介绍一下把创建的Python虚拟环境设置成本地系统环境。首先我单击菜单栏的file按钮，找到设置settings，单击settings

在弹出的窗口中展开Project：StudyPython，单击Project Interpreter 在右侧的面板中，我们单击右上角的三角号，在下拉菜单中单击Show All按钮 在弹出的窗口中我们单击右侧的+号 ![28单击+号](/Users/liuyang/Desktop/Python学习笔记/02 Python 搭建环境/04Windows安装Pycharm/压缩后/28单击+号.jpg) 在弹出的窗口中我们选择系统解释器，解释器路径为我们安装Python的路径 在弹出的窗口中单击OK按钮 在弹出的窗口中单击Apply按钮，然后单击OK按钮，这样我们就把Python虚拟环境设置成本地系统环境了
